Linear Search

Description

Searches input array one element at a time, checks each time if arr[i] === target.

Time Complexity: \(O(n)\)

Code Example

function linear_search(data: number[], target: number): boolean {
    for(let i = 0; i < data.length; ++i) {
          if (data[i] === target) { 
              return true;
          }
      }
      return false; 
    }
