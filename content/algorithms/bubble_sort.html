Bubble Sort

Description

Takes an element at a time from an unsorted array. Iteratively compares to the element to right and swaps if higher.

Time Complexity: \(O(n^{2})\). Why? Because the worst case here is the list sorted in reverse order. Here we would perform n-1 passes with n-1, n-2, n-3 .... 1 comparisons. Therefore we have \(\frac{n(n-1)}{2}\) comparisons (it's an arithmetic series).

Code Example

function bubble_sort(data: number[]): boolean {
    let swapped = false;
    do {
        swapped = false;
        for (let i = 0; i < data.length; i++) {
            if (data[i] > data[i+1]) {
                const temp = data[i];
                data[i] = data[i+1];
                data[i+1] = temp;
                swapped = true;
            }
        }
    } while (swapped);
